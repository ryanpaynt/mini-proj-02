CSS Snippet Cheat Sheet

A cheat sheat full of my favorite CSS codes.

Media Rule

The @media rule is used in media queries to apply different styles for different media types/devices. Such as phones and different monitor sizes.

@media screen and (max-width: 768px){
        header{
                
        }
    }
Transform Translate

The transform element is super useful and enables the elements of a page to be moved, skewed, and rotated.

   
header{
    tramsform: translateY(0%)
}
Hover Pseudo Code

The hover element is part of the pseudo code class. It enables the developer to change the look of webpage elements once the cursor is hovering over them.

footer:hover{
        color: green;
}
Before/After Elements

This pseudo-element can be used to insert some content before the content of an element.

@media screen and (max-width: 768px){
    p::before{
        content: smiley face
        }
Float Positioning

This CSS element focus on the positioning of things in your webpage. Left or right float flows with the text around it when used.

footer{
    float: left;
}
Shadow Boxing

Shadow boxing is used to create depth with your box borders

header{
    box-shadow: 5px 10px 10px gray;
}
